Duration

Allow approximately 60 minutes to complete this module.

Main menu

You can navigate to the main menu via the hamburger menu in the top right.

Exiting the module

Do not exit the module by closing your browser. If you need to exit, go to the menu and select Save & exit to bookmark your location. When you return, you can pick up from where you left off.

External links

Through-out this module you will be asked to access external web links. When you select these links, they will open in a new window. This module will remain open in this window.

Listen

This module contains activities that include video. Have headphones ready if you’re not in a quiet place.

Your responses

Through-out this module you will be asked to enter reflections on your own practice. Your responses will be collated into a PDF you can download at the end of the module.

Completion

Finish every activity to complete this module.
